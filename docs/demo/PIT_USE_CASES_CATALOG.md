# CATALOGUE DES CAS D'USAGE MÉTIER ENRICHIS (PIT_USE_CASES_CATALOG)

Ce document décrit les 15 cas d'usage réels et cohérents servant de base de données démonstratrice pour la PIT vNext. Ils racontent des histoires concrètes de transformation d'entreprises wallonnes accompagnées par les acteurs du territoire, intégrant désormais la gouvernance des parties prenantes et l'alignement stratégique.
