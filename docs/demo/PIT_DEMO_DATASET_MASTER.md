# PIT Master Demo Dataset 2.0 (Premium)

Ce document décrit le jeu de données démonstrateur de la PIT. Toutes les entités et relations présentées ci-dessous forment des récits cohérents illustrant l'accompagnement d'innovation territoriale de la Wallonie.

---

## 🏭 Les 6 Filières Stratégiques S3 Régionales

### 1. Santé (BioWin)
- **Acteurs clés** : MedTech Namur, CHU Liège, UCLouvain, GSK, IBA, Coris BioConcept, Diagenode.
- **Communautés** : IA Santé, Biotech Wallonie.
- **Chaîne de valeur** : Santé Numérique (e-Santé).
- **Verrou technologique (gaps)** : Absence de PME d'essais cliniques qualifiées NIS2, manque d'accompagnement marquage CE MDR pour logiciels IA.

### 2. Hydrogène Vert (GreenWin / MecaTech)
- **Acteurs clés** : HydroGreen, Sirris, Cockerill, John Cockerill Hydrogen, Fluxys, Aperam, AGC.
- **Communautés** : Hydrogène Wallon, Décarbonation.
- **Chaîne de valeur** : Vecteurs Énergie.
- **Verrou technologique (gaps)** : Absence de stockage 700 bars local homologué, manque d'experts certifiés en intégration IA de prédiction de pression.

### 3. Agroalimentaire (Wagralim)
- **Acteurs clés** : AgriFood Solutions, Gembloux Agro-Bio Tech, Cosucra, Puratos, Belourthe, Keyfood.
- **Communautés** : AgriTech, Transition Alimentaire.
- **Chaîne de valeur** : Protéines végétales.
- **Verrou technologique (gaps)** : Logistique d'approvisionnement ultra-locale non structurée, manque d'emballages bio-sourcés industriels.

### 4. Mobilité & Fret (Logistics in Wallonia)
- **Acteurs clés** : LogiTrans, SmartFleet, DataMove, Port Autonome de Liège, TEC, Eurogentec.
- **Communautés** : Smart Mobility, Cyber Fret.
- **Chaîne de valeur** : Fret Vert.
- **Verrou technologique (gaps)** : Faiblesse NIS2 des systèmes de dispatch PME, interopérabilité des plateformes de routage.

### 5. Construction (Circular Wallonia)
- **Acteurs clés** : CCW (Confédération Construction), Tradecowall, Recy béton S.A., CBR.
- **Communautés** : Bâtiment Vert, Éco-Matériaux.
- **Chaîne de valeur** : Béton Circulaire.
- **Verrou technologique (gaps)** : Tri de granulats béton non automatisé, manque d'éco-design.

### 6. Numérique (Digital Wallonia)
- **Acteurs clés** : CETIC, UCLouvain, Multitel, Sirris, Odoo, EVS, N-Side.
- **Communautés** : Cyber PME, IA & Data Space.
- **Chaîne de valeur** : Data Space Wallonie.
- **Verrou technologique (gaps)** : Pénurie critique de data scientists et d'architectes cloud NIS2 certifiés.

## 👥 Récit Master 1 : BioWin e-Santé
- **Entreprise** : MedTech Namur (Startup d'imagerie clinique).
- **Défi** : Obtenir la certification CE MDR de classe IIa pour un logiciel IA de détection de tumeurs.
- **Parcours** : IA Santé (Initiation du conseiller).
- **Services mobilisés** :
  1. Diagnostic IA clinique (SRV-IA-DIAG) - Délivré par BioWin.
  2. Test Before Invest (TBI) - Accès aux clusters de calcul du CETIC.
- **Consortium** : MedTech Namur, UCLouvain (Labo Image), CHU Liège (Validation Clinique).
- **Financement** : Appel Health Innovation 2026 (Horizon Europe) - 2.4 M€.
- **Projet** : AI Diagnostics.
- **Outcome** : Algorithme clinique certifié MDR, amélioration de 25% de la précision.
- **Preuve (Evidence)** : Rapport clinique officiel du CHU Liège validé (status: APPROVED).
- **Contributions** : S3 Santé, Digital Wallonia.

## 👥 Récit Master 2 : GreenWin / MecaTech Hydrogène
- **Entreprise** : HydroGreen S.A.
- **Défi** : Concevoir des réservoirs de stockage hydrogène à 700 bars avec sécurité prédictive.
- **Parcours** : Industrie 5.0 / Transition Hydrogène.
- **Services mobilisés** :
  1. Plateforme de test de pression (SRV-TBI-HYDRO) - Sirris.
  2. Audit transition énergétique - GreenWin.
- **Consortium** : HydroGreen, Sirris, John Cockerill Hydrogen.
- **Financement** : Appel Décarbonation Plan de Relance (PRW) - 1.5 M€.
- **Projet** : HydroSeraing.
- **Outcome** : Prototype d'électrolyseur et réservoir haute pression homologué.
- **Preuve (Evidence)** : Attestation de déploiement de capteurs prédictifs IA validée (status: APPROVED).
- **Contributions** : Plan de Relance (PRW), Green Deal.

## 👥 Récit Master 3 : Logistics Cyber NIS2
- **Entreprise** : LogiTrans S.A.
- **Défi** : Sécurisation du dispatching fret NIS2 contre les ransomwares.
- **Parcours** : Cyber PME (Initiation conseiller).
- **Services mobilisés** :
  1. Cyber Diagnostic (SRV-CYBER-DIAG) - CETIC.
  2. Remédiation NIS2 - Forem.
- **Consortium** : LogiTrans, CETIC, SmartFleet.
- **Financement** : Chèques Cybersécurité Wallonie Entreprendre - 50 k€.
- **Projet** : Fret Vert Connecté.
- **Outcome** :Dispatching chiffré certifié résilient.
- **Preuve (Evidence)** : Rapport final d'audit NIS2 du CETIC approuvé (status: APPROVED).
- **Contributions** : Digital Wallonia.
